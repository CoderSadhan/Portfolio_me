# 💻 Sadhan Das — Portfolio Website

Welcome to my personal portfolio website built with **Next.js**, **React.js**, and **modern UI design** principles. It showcases my work, projects, skills, and journey as a full stack developer passionate about **Web Development and modern frontend/backend technologies**.

🌐 **Website:** [sadhan-das-portfolio](https://sadhandas.vercel.app/)

---

## 🚀 Features

- 🎨 **Modern UI/UX** with dark-based neon glassmorphism aesthetic theme
- ⚡ Smooth **micro-animations** and hover transitions
- 💼 **Projects, Interactive Skills, Experience, and Contact sections**
- 📨 Functional contact form powered by **EmailJS**
- 🧩 Fully responsive layout optimized for all device widths

## ⚙️ Setup Instructions

To run this project locally:

```bash
# 1️⃣ Clone the repository
git clone https://github.com/CoderSadhan/Portfolio.git

# 2️⃣ Navigate to project directory
cd My_Portfolio

# 3️⃣ Install dependencies
npm install

# 4️⃣ Run development server
npm run dev
```

Now open [http://localhost:3000](http://localhost:3000) in your browser 🚀
